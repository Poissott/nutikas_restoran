# Nutikas Restorani Reserveerimissüsteem

Kõigepealt alustasin ma backendiga, luues põhilised entity'd - table ja reservation. Ning seejärel lõin ma mõlemale Spring Bootile omase tüüpilise struktuuri. 
Selle osa põhimõtte kirjutasin ma ise, kuid samas on seal väga palju kordusi (palju korduvaid CRUD ja REST operatsioonide elemente), mistõttu pidasin loogiliseks kasutasda seal AI genereerimist, et need kordused saaks kiiremini kirjutatud (kuid loomulikult valideerisin ma igat genereeritud rida isiklikult).
